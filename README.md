## PS1 - Image Processing (CamPaint)

Java webcam-based painting project. It uses region finding to recolor frames and lets you “paint” by selecting a target color from the live video.

### Run
- Primary entry point: `src/CamPaint.java`
- Debug/testing: `src/RegionsTest.java`

If you run `CamPaint`, make sure macOS camera permissions are allowed for this Java app.

### Notes
- This project is GUI/video driven (webcam). It does not require input data files.
